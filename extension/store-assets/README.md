## How these were made / how to regenerate

The `.html` files are the editable source. Each renders the **real** popup styling
(`../src/popup/popup.css`) and the **real** eye icon (`../icons/icon128.png`) on a branded
canvas sized exactly to the asset, so they must be served from the extension root:

```bash
cd extension
python3 -m http.server 8903           # serve the extension root
# then, for each asset, in a headless/devtools Chrome:
#   navigate to  http://localhost:8903/store-assets/<name>.html
#   set the viewport to the asset's exact CSS size
#   capture a full-page PNG (it comes out at 2× on a retina display)
#   downscale to the exact size, e.g.:
sips -z 800 1280 raw.png --out screenshot-1-results.png   # -z is height width
sips -z 280 440  raw.png --out promo-small.png
```

Edit the copy/colours in the HTML and re-export. Fonts (Fraunces + Hanken Grotesk) load from
Google Fonts **for rendering only** — they are not part of the shipped extension.
